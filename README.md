## Паттерн Memento (Хранитель, Снимок)

**Для демонстрации!**


### Классы:
- **Originator**: Класс создателя, хранящий в себе своё текущее состояние. Содержит метод *save_state* для создания снимка и метод *restore_state* для отката до предыдущего состояния
- **Memento**: Класс снимка, содержащий все поля класса создателя. Предназначен для хранения состояния полей объекта создателя
- **Caretaker**: Класс опекуна, хранящий последний снимок объекта создателя. В идеале должен хранить большее количество снимков, но для базовой демонстрации хранит только одно последнее состояние
